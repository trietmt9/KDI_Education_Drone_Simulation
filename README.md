# Drone Dashboard

A real-time 3D drone visualization and telemetry dashboard built with PySide6 and Qt Quick 3D.

## Features

- **Real-time Telemetry Display**: Live monitoring of drone orientation data (roll, pitch, yaw)
- **3D Drone Visualization**: Interactive 3D model of the drone with real-time orientation updates
- **Modern UI**: Sleek dark-themed interface with gradient backgrounds and animated status indicators
- **Connection Status**: Visual connection indicator with animated pulse effect

## Technology Stack

- **Backend**: Python with PySide6
- **Frontend**: Qt Quick (QML) with Qt Quick 3D
- **3D Models**: Custom drone meshes and materials
- **Real-time Updates**: Timer-based data simulation (easily replaceable with real sensor data)
